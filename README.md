# OSPBL_KeyValueStore

## Prerequisite

* cmake (minimum required 3.5.1)
* [HomeBrew](https://brew.sh/index_ko.html) (Mac optional) (if you have to install cmake from brew)

## Build

For Mac
```
$ brew install cmake
$ mkdir build && cd build
$ cmake ..
$ make
$ ./main
```

For Ubuntu
```
$ apt-get install cmake
$ mkdir build && cd build
$ cmake ..
$ make
$ ./main
```
